# Hunain Command Center

## Setup
1. Install Node.js (v18+) if not installed
2. Run: `cd backend && npm install`
3. Run: `./start.sh` or `cd backend && node server.js`
4. Open: http://localhost:3000

## Features
- 🌐 Homepage with 3D globe, real-time clock & weather (Brussels)
- 💪 Body page: weight tracking, nutrition, weekly workout plan
- 🎓 University: 4 subjects with all topics, slides & labs
- 💰 Wealth: 611 ICT videos across 32 playlists with note-taking

## Data Persistence
All progress saves to `backend/tracker.json` automatically.
